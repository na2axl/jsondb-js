Manage JSON files as databases with JSONDB Query Language (JQL)

Performance

Data are stored locally, then retrieved more quickly than a distant database.

JQL

Manage all your databases with JQL, a query language designed for JSONDB.

PDO-like syntax

Don't want to learn another boring syntax? Don't worry, use JSONDB like PDO!

Do more with a .json file...

var JSONDB = require('jsondb-js');
var jdb = new JSONDB();
var database = jdb.connect('server_name', 'username', 'password', 'db_name');
var results = database.query("table_name.select(*).where(foo = 'hello', bar = 'world')");
while (result = results.fetch()) {
    console.log(result.foo + ' ' + result.bar);
}
database.disconnect();
